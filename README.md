Appscrip Assignment

This is a fully responsive Product Listing Page (PLP) built as a frontend assignment using Next.js and pure CSS (no frameworks).

Features

-Responsive design (Desktop, Tablet, Mobile)
-Filter sidebar UI
-Product grid layout
-Wishlist 
-Clean and structured UI 

Key Highlights

-Built without any CSS framework (pure CSS)
-Mobile-first responsive design
-Clean component-based architecture
-Focus on pixel-perfect UI
-Optimized layout and spacing


Tech Stack

-Next.js (React)

-JavaScript (ES6)
